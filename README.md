# 9V to 5V DC-DC Buck Converter using IR2110

## Overview
This project implements a PWM-controlled DC-DC Buck Converter that converts a 9V DC input into a regulated 5V output using an N-Channel MOSFET driven by the IR2110 gate driver IC.

The converter is designed to drive a 5V DC motor and supports both full-speed and half-speed operation by varying the PWM duty cycle.

## Features
- 9V to 5V DC voltage conversion
- PWM-based speed control
- IR2110 gate driver for efficient MOSFET switching
- 260 µH inductor-based buck topology
- 1N5819 Schottky freewheeling diode
- Suitable for embedded and battery-powered applications

## Components Used
- ESP8266 / Arduino
- IR2110 Gate Driver
- N-Channel MOSFET
- 1N5819 Schottky Diode
- 260 µH Inductor
- 22 µF Capacitor
- 5V DC Motor

## Working Principle
The converter operates by switching the MOSFET using PWM signals generated by the microcontroller.

Output voltage is controlled by adjusting the duty cycle:

- 55.6% Duty Cycle → 5V Output (Full Speed)
- 27.8% Duty Cycle → 2.5V Output (Half Speed)

## Results
The circuit successfully stepped down a 9V input to approximately 5V and powered a 5V DC motor with good efficiency.
